## Installation
### Vim

```sh
$ curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

$ cd dotfiles
$ mv vimrc $HOME/.vimrc
```

### Bash

```sh
$ cd dotfiles
$ mv aliases $HOME/.aliases
$ mv colorbash $HOME/.colorbash
$ cat bash_profile >> $HOME/.bash_profile
```
